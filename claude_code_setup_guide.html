🖥 Cursorのターミナルを開く

Cursorの画面の中に「ターミナル」という黒い入力画面を出します。

  1. Cursorを開いた状態で、キーボードの Ctrl + ` を押す
    （` は数字 1 の左にあるキー。Shiftは押さない）
  2. 画面の下半分に黒い画面が出る → これが「ターミナル」

ターミナルが出ない場合：Cursor上部のメニュー「Terminal」→「New Terminal」をクリック

Git・Node.js・Claude Code を全部まとめてインストールします。

winget install --id Git.Git -e --source winget; winget install OpenJS.NodeJS.LTS; npm install -g @anthropic-ai/claude-code

🔴 こんなエラーが出たら

'git' is not recognized as an internal or external command
'node' is not recognized as an internal or external command

= 「パスが通っていない」エラー。PCが「そのプログラムどこ？」と迷ってる状態。

🔧 順番に試す

STEP 1: Cursorを再起動

右上の×で完全に閉じる → もう一度開く → ターミナル開く → コマンド再実行

↓ ダメなら

STEP 2: PC再起動

スタートメニュー → 電源 → 再起動。パスが反映される。

↓ ダメなら

STEP 3: パスを手動で通す（PowerShell）

$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")
↓ ダメなら

STEP 4: 環境変数を手動設定

  1. Windowsの検索バーに「環境変数」と入力
  2. 「環境変数を編集」をクリック
  3. 「Path」を選んで「編集」
  4. 「新規」で以下を追加：
C:\Program Files\Git\cmd
C:\Program Files\nodejs

Step 3: ツール一括インストール

Cursorのターミナル（ Cmd + ` で開く）に以下を順番に貼り付けて Enter：

① Homebrew（まだ入ってなければ）

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

② Node.js

brew install node

③ Claude Code

curl -fsSL https://claude.ai/install.sh | bash
✅
MacはGitが最初から入ってるのでインストール不要。Windowsより簡単です。

claude で起動 → ログイン → Chrome拡張 → /chrome はWindowsと同じ手順です。
